Portable Quiz Package
=====================

This folder is self-contained and can be moved anywhere.

Files
-----
- index.html        -> Main quiz page
- quiz.css          -> Styling/branding
- quiz-content.js   -> Quiz questions/content (editable)
- quiz-app.js       -> Quiz behavior/logic
- logo.svg          -> Logo asset
- quiz-ipad.html    -> iPad-ready single-file version (best for transfer)

How to use
----------
1) Move/copy the entire "quiz-package" folder anywhere.
2) Open index.html directly by double-clicking it.

No server required.

iPad mode (recommended)
-----------------------
Use "quiz-ipad.html" for iPad. It is fully self-contained:
- no external files
- no local path dependencies
- no server required

How to run on iPad:
1) AirDrop "quiz-ipad.html" to iPad.
2) Save it in Files (On My iPad).
3) Tap and hold -> Share -> Open in Safari.
4) Optional: Safari Share -> Add to Home Screen for kiosk-style launch.
